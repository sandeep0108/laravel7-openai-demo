# Laravel 7 OpenAI Demo (PHP 7.4)

Simple OpenAI ChatGPT integration for **Laravel 7** and **PHP 7.4**.

## Requirements
- PHP 7.4
- Laravel 7
- Composer

## Setup

```bash
git clone https://github.com/sandeep0108/laravel7-openai-demo.git
cd laravel7-openai-demo
composer install
cp .env.example .env
php artisan key:generate
```
